
# Introduction To Full Stack Development

This repository contains my implementations and exercise solutions for the **[Introduction To Full Stack Development](https://opiskelijanopas.tuni.fi/en/tampere-university/curriculum/course-units/otm-949208ad-9f08-452d-bd64-78588079c3d6?year=2025)** course completed at Tampere University.

The course focuses on designing and building simple end-to-end web applications while understanding core web technologies and architectural concepts.  

![cover](https://github.com/theanasuddin/Introduction-To-Fullstack-Development/blob/master/cover.png)

---

## Overview

The repository demonstrates practical implementation of:

- Frontend fundamentals
- Backend development with Node.js
- RESTful APIs
- Authentication and security
- Data persistence
- MVC architecture principles

All exercises are implemented using modern JavaScript and backend technologies.

---

## Topics Covered

### 1. HTML & DOM
- Semantic HTML
- Forms, tables, metadata
- DOM manipulation fundamentals

### 2. CSS & Responsive Design
- CSS Grid
- Media queries
- Accessibility principles

### 3. JavaScript
- DOM tree traversal
- Event handling
- JSON messaging
- Functional programming concepts

### 4. HTTP & Client-Server Architecture
- HTTP requests & headers
- Node.js server creation
- REST principles
- Routing & API testing

### 5. Templating, Cookies & Sessions
- Cookie handling
- Server-side sessions
- Express.js fundamentals

### 6. Asynchronous JavaScript
- Callbacks
- Promises
- Async/await
- Mocha testing

### 7. Security
- CORS
- CSP
- CSRF
- XSS prevention basics

### 8. Authentication & Authorization
- User registration
- Session-based auth
- ESLint & code quality practices

### 9. Data Persistence
- LocalStorage
- MongoDB basics
- Mongoose schemas
- Secure database connections

### 10. Architecture & Organization
- MVC architecture
- Controllers & middleware
- Configuration management
- Application extensibility
